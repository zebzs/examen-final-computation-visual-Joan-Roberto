## Descripción general

Este repositorio contiene la solución al examen final de Computación Visual. Se desarrollan dos ejercicios:

1. **Ejercicio 1 – Procesamiento visual e IA**: Aplicación en Python que carga una imagen/video, aplica transformaciones (escala de grises, otro espacio de color, suavizado, detección de bordes, segmentación/detección) y guarda resultados comparativos.
2. **Ejercicio 2 – Escena 3D interactiva temática**: Escena 3D desarrollada con [Three.js / React Three Fiber / Unity] sobre el tema “[tema elegido]”, incluyendo jerarquías, transformaciones, materiales PBR, iluminación, animaciones e interacción.

## Dependencias

### Ejercicio 1

- OpenCV (`pip install opencv-python`)
